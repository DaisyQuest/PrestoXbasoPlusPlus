# Xbase++ Samples

This directory contains sample Xbase++ programs used to validate navigation, structure view,
and formatting behaviors in the Presto Xbase++ IntelliJ plugin.

## Files
- `hello_world.xb`: entry-point functions, procedures, and control flow.
- `inventory.prg`: a multi-function module with shared state, loops, and data manipulation.
- `ui_workflow.xb`: UI-oriented helper routines and string handling.
