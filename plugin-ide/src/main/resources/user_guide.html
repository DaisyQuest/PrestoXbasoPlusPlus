Presto Xbase++ Plugin User Guide

Welcome to the Presto Xbase++ plugin for IntelliJ IDEA. This guide highlights the language features supported by the plugin, explains how to use core IDE tools, and describes the available settings.

Getting Started

  1. Install the plugin from the IntelliJ IDEA plugin settings.
  2. Open or create a file with the .xb or .prg extension (the plugin registers both file types automatically).
  3. Start typing xbase++ code to activate syntax highlighting, inspections, completion, navigation, and structure view.
  4. Explore the /samples folder in this repository for rich, working examples.

Language Features

Syntax Highlighting

  • Keywords for control flow and logic (IF, ELSE, WHILE, RETURN).
  • Operators, punctuation, string literals, and numeric literals.
  • Explicit error highlighting for invalid or unterminated tokens.
  • Highlighting is provided by the IntelliJ syntax highlighter factory wired to the .xb and .prg file types.

Diagnostics and Error Annotations

  • Unexpected characters and malformed tokens are annotated with descriptive messages.
  • Unterminated string literals are reported with clear recovery diagnostics.
  • Inspections warn about empty statements, redundant parentheses, constant conditions, self-comparisons, and unreachable statements.
  • Inspection severities distinguish between errors, warnings, and informational hints.

Code Completion

  • Completion suggestions for function and variable declarations in scope.
  • Optional case-insensitive matching for rapid navigation.

Navigation and Usages

  • Go to Definition works from any identifier under the caret in .xb or .prg files.
  • Find Usages is supported from function and variable declarations.
  • Navigation targets are resolved from the lexed PSI snapshot so symbols inside a file are immediately discoverable.
  • Function, procedure, and method declarations participate in the same navigation index.

Structure View and Breadcrumbs

  • Structured outline of file-level declarations (functions, procedures, methods, variables).
  • Breadcrumbs to understand nesting and scope at a glance.
  • Click a structure view item to jump directly to the declaration.

Expanded Macro Panel

  • Open the Xbase++ Macros tool window to review expanded #define directives.
  • See raw and expanded values side-by-side, including unresolved or recursive references.
  • The panel updates as you switch between .xb and .prg files.

Formatting and Folding

  • Indentation-aware formatter for IF/ELSE/ENDIF and loop blocks.
  • Foldable regions for control flow blocks to reduce visual noise.

Rename Refactoring

  • Rename symbol declarations and update connected references.
  • Preview renamed symbols before applying changes.

Settings

The plugin exposes a settings panel for formatting and inspection preferences. Settings are persisted per IDE configuration and can be reset at any time.

Tip: If you are evaluating the plugin in a test project, keep sample files small to explore completions, navigation, and structure view results quickly.

Samples

The repository ships a /samples directory with Xbase++ programs that exercise the language features and IDE tooling. Open any file to explore navigation and structure view.

  • samples/hello_world.xb: basic functions, procedures, and control flow.
  • samples/inventory.prg: multi-function module with shared data and loops.
  • samples/ui_workflow.xb: UI-oriented helper routines and string handling.

Troubleshooting

  • If highlighting or completion seems unavailable, ensure the file extension is .xb or .prg and the file is not treated as plain text.
  • Use the IDE logs to report any unexpected parsing errors.
  • Restart the IDE after installing the plugin so the file type registration is refreshed.
